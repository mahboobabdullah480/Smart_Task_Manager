📝 Smart Task Manager (Flutter)
A modern, secure, and high-performance task management application built with Flutter. This project was designed with a focus on User Privacy and Offline-first capability, making it a reliable tool for daily productivity.
✨ Key Features
🔒 Biometric Authentication: High-level security using Fingerprint/FaceID to protect your tasks.

🚀 Offline-First: Powered by Hive (NoSQL) database for lightning-fast performance without needing an internet connection.

📊 Priority Management: Organize tasks by priority levels (High, Medium, Low) with visual indicators.

📱 Responsive & Clean UI: Fully optimized for mobile devices with a smooth user experience.

🌍 RTL Support: Designed to support Right-to-Left languages (like Arabic) with professional fonts (Cairo/Tajawal).
🛠️ Tech Stack
Tool                             Purpose
Flutter/Dart            Cross-platform mobile framework
Hive                    Local NoSQL database for data persistence
Local Auth              Biometric authentication (Fingerprint/FaceID)
Intl                    Date formatting and localization
